# shch-poc

#### 介绍
上清所POC工程

#### 工程结构
1.shch-common   支持演示的基础组件,微服务中已经默认引入依赖

2.shch-bond 簿记服务
    shch-bond-client     客户端模块
    shch-bond-core       业务模块
    shch-bond-starter    启动模块
    
3.shch-cash 资金服务
    shch-cash-client   客户端模块
    shch-cash-core     业务模块
    shch-cash-starter  启动模块
    
4.shch-clear  清算服务
    shch-clear-client     客户端模块
    shch-clear-core       业务模块
    shch-clear-starter    启动模块
